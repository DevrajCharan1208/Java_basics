# Java Basics

A repository for foundational Java programming exercises, assignments, and problem-solving scripts. 

## Contents

This repository includes Java programs covering the following core concepts:

* **Object-Oriented Programming:** Implementation of classes, objects, and multilevel inheritance (e.g., banking details systems).
* **Control Flow & Logic:** Scripts utilizing loops and conditional statements, such as prime number checkers.
* **Math & Syntax:** Foundational Java syntax, data types, and mathematical formulas like area calculations.

## Requirements

* Java Development Kit (JDK) 8 or higher
